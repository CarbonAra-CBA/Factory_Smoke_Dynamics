앱 개발 요구사항

- 개인 정보 수집 컴포넌트
- 한줄 정보 표기 컴포넌트
- 소주제 제목 컴포넌트
- 지도 컴포넌트
- 드롭다운 컴포넌트
- 여러 줄 정보 표기 컴포넌트
- 시간별 건강정보 표기 컴포넌트(표 형식)
- 보고서 평가 결과 정보 표기 컴포넌트

앱 한줄 설명 : 인근 공장에서 배출한 매연의 확산이 사용자의 건강에 영향을 주는지 알려주는 보고서 작성 프로그램

1. 데이터 수집 및 정보 활용 동의:
<수집할 데이터>
- 사용자 거주기간
- 사용자 거주 위치
- 이름
- 생년월일
- 사용자 건강 데이터
위 데이터를 수집하는 ui 작성
<개인정보 수집 동의 항목>
- 개인정보 활용 동의서(주거지, 이름, 생년월일, 건강정보)
위 동의서를 나타내는 ui 작성

2. 지난 한달간 인근 공장에서 사용자의 거주지로 흘러 들어온 오염물질 스냅샷 보여주기
<지도 상 표현 항목>
- 사용자 거주지 위치(마커)
- 인근 공장들 위치 중 택 1(마커)
- 공장에서 거주지로 흘러 들어오는 오염물질의 밀도를 히트맵으로 표현하기
<공장에서 나오는 오염물질 양 표시>
- 지도 하단에 공장에서 가장 많이 배출된 오염물질을 1개 선정
- 해당 오염물질이 얼마나 인체에 유해한지 설명
- 해당 오염물질이 거주지에 확산된 양을 표현
<지도 좌우 화살표 버튼>
- 좌 또는 우를 누를 때마다 거주지 인근 공장들 중에서 지도 상에 표시할 공장들을 교체
- 또한 하단에 오염물질 양 표시도 변경
<유료 보고서 작성 버튼>
- 위 무료 제공 정보를 통해 위험성을 확인한 소비자 중 더 정밀한 건강영향평가 보고서를 출력하고 싶은 소비자가 누르는 버튼. 보고서 생성 로딩 창으로 이동한다.

3. 보고서 생성 로딩 창
- 사용자의 질환과 노출된 오염물질 간의 상관관계를 분석하는 동안 프로그래스 바를 보여준다.
- 분석이 끝나면 최종 보고서 창으로 넘어가게 된다.

4. 최종 보고서 
<연도별 사용자 질환 대비 오염물질 배출량 비교 표>
- 사용자의 질환과 관계있는 오염물질이 질환 발병 이전부터 꾸준히 배출되어 거주지로 확산되었다면 오염물질로 인해 발병했을 확률이 높으므로 높은 점수를 준다.
- 그렇지 않다면 낮은 점수를 준다.
- 각 질환별로 점수를 매길 수 있다.
- 0~100점 범위의 점수를 가진다.
- 원한다면 사용자는 월별로 오염물질 배출량과 질환을 비교할 수 있는 표를 접었다 폈다 할 수 있다(표를 상세히 보거나 표를 생략할 수 있다)

21:50 ~ 22:10 : 만든 요구사항으로 웹앱 UI 만들어보기
